What's A.I. Challenge @ CODE FESTIVAL?

A.I. Challengeは、CODE FESTIVALに併設されたゲームAIプログラミングコンテストです！

我々AI-COMPは、2014年度をゲームAIプログラミングコンテスト普及の年と銘打ち、
本コンテストに加えCEDEC AI CHALLENGE、JavaChallengeを主催、SamurAI Codingの開催に協力しています。
各コンテストでは、選手の方が複数コンテストに参加しやすいように類似ルールを利用しています。

本コンテストはJavaChallengeのテーマを受け継いだ続編になります。
また、本コンテストとSamurAI Codingは同じ類似ルールを使用しますので、奮ってご参加下さい！

CODE FESTIVAL本戦参加者とオンライン参加者のそれぞれ上位2名には素敵な賞品も用意しました！

For Beginners

ゲームAIプログラミングコンテストは主催者が提供するゲーム（例えば囲碁や将棋など）に対して、
参加者がコンピュータプレイヤーを操作するAIプログラムを作ることで競い合うコンテストです。
ゲームをプレイする感覚でプログラミングを楽しめるため、初学者向けの教材として注目を浴びています。

本コンテストでは、将棋や囲碁よりもシンプルで同じくらい奥深いルールのゲームを提供しています。
プログラミングを始めたばかりのあなたでも楽しめる内容となっていますので、是非ご参加ください！
